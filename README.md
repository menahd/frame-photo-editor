# Frame Photo Editor

A lightweight web application to create profile photos with a decorative frame overlay, interactive cropping, and high-resolution PNG export.

## Features

- 📸 Import photo or take a selfie (mobile/desktop)
- 🖱️ Drag to reposition, zoom with +/- buttons or pinch gesture
- 🎨 Overlay a decorative frame with transparency support
- 📥 Export as 1836×1836 PNG (high resolution)
- 📱 iOS Web Share support

## Quick Start

1. Open `index.html` in a modern web browser
2. Choose a photo or take a selfie
3. Adjust position and zoom as needed
4. Click Download to export your framed photo

## Customization

To use a different frame:
1. Replace `frame.png` with your custom frame image
2. Update the `FRAME_FILE` variable in the code if needed

## Browser Support

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)
